# COMBO-UI Editor

COMBO-UI se compone de un editor de componentes UI y varios paquetes npm que permiten utilizar susw2 diseños de temas en tu App frontend y ver los cambios aplicados en tiempo real.

1. Creas el tema en el editor.
2. Instalas COMBO-UI en tu app como paquete npm.
3. Creas la páginas de tu App.
4. Cambia el diseño del tema mientras ves los cambios en tu propia app.
